Hardware Overview

Collect CPU, memory, and disk details.

@echo off
color 0A
echo === Hardware Overview ===
wmic cpu get Name,NumberOfCores,NumberOfLogicalProcessors
wmic MEMORYCHIP get Capacity,Speed,Manufacturer
wmic diskdrive get Model,Size,SerialNumber
pause

Event Logs Snapshot

Export recent system event logs.

@echo off
color 0A
echo === Event Logs Snapshot ===
powershell -Command "Get-EventLog -LogName System -Newest 100 | Format-Table -AutoSize" > system_logs.txt
echo Saved to system_logs.txt
pause

Network Report

List adapters, IPs, and routes.

@echo off
color 0A
echo === Network Report ===
ipconfig /all
route print
netstat -ano | findstr :80
pause
