CPU Load Simulation

Generate sustained CPU load for profiling.

@echo off
color 0A
echo === CPU Load Simulation ===
for /l %%x in (1, 1, 4) do start /b powershell -Command "while($true) { [Math]::Sqrt(12345) }"
timeout /t 20
Taskkill /F /IM powershell.exe
pause

Network Latency Simulation

Introduce artificial delays in requests.

@echo off
color 0A
echo === Network Latency Simulation ===
for /l %%x in (1,1,10) do (
  ping 8.8.8.8 -n 1 >nul
  timeout /t 1
)
pause

Disk IO Test

Write and read a test file repeatedly.

@echo off
color 0A
echo === Disk IO Test ===
set FILE=%TEMP%\io_test.txt
for /l %%x in (1,1,1000) do echo %%x >> %FILE%
type %FILE% >nul
del %FILE%
pause
