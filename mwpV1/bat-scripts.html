Network Diagnostics

Flush DNS, renew IP, and test connectivity.

@echo off
color 0A
echo === Network Diagnostics ===
ipconfig /flushdns
ipconfig /release
ipconfig /renew
ping 8.8.8.8 -n 4
tracert 8.8.8.8
pause

System Cleanup

Remove temp files safely.

@echo off
color 0A
echo === System Cleanup ===
set TEMP_DIR=%TEMP%
rd /s /q "%TEMP_DIR%"
md "%TEMP_DIR%"
cleanmgr /sagerun:1
pause

Process Snapshot

Export running processes to a timestamped file.

@echo off
color 0A
set LOG=processes_%DATE:~-4%%DATE:~4,2%%DATE:~7,2%_%TIME:~0,2%%TIME:~3,2%.txt
wmic process get name,ProcessId,WorkingSetSize /format:list > %LOG%
echo Saved to %LOG%
pause
